# Random Student Picker

A beginner-friendly Python project that randomly selects one student from a list of four names provided by the user.

## Features

- Accepts four names as input
- Stores the names in a list
- Randomly selects one name
- Displays the selected name

## Technologies Used

- Python 3
- Built-in `random` module

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/random-student-picker.git
```

2. Navigate to the project folder:

```bash
cd random-student-picker
```

3. Run the script:

```bash
python main.py
```

## Example

### Input

```text
First name: Alice
Second name: Bob
Third name: Charlie
Fourth name: Diana
```

### Output

```text
The selected student is Charlie
```

## Learning Objectives

This project demonstrates:

- User input handling
- Working with lists
- Importing and using Python modules
- Random selection from a collection
- Basic variable manipulation
